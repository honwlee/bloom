### bloom
班级聚会网站

### 功能
* 用户登录注册
* 用户信息配置
* 班级信息管理
* 联络簿
* 精彩内容
* 聚会照片展示
* 聚会倒计时与邀请函

### start
* $ git clone https://github.com/honwlee/bloom.git
* $ cd bloom; npm install; npm run dev;

### build
* $ cd bloom/build; npm install;
* $ gulp sass;
